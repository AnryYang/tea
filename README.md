## How to use
#### Requirements
- GNU Linux system
- GCC 4.8
- Boost library

#### Compile
```sh
$ cmake .
$ make clena all -j4
```

#### Run
```sh
$ ./hk -a teaplus -f ./data/ -g dblp -n 50 -s seeds.txt
```
